code-server
----
this script/programme help to make easy to use code-server in android/termux.

## installtion

- step 1 
```
 git clone https://github.com/rrkroms/code-server.git
```
- step 2
```
bash ./install.sh -i
```

## uninstalltion

```
bash ./install.sh -r
```
> Note : use -y as $3 for pre-confession for uninstalltion
